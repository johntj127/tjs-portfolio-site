# Claude Build Brief — Portfolio Map SNES Overworld Direction

## Goal
Build the portfolio map so it feels like a **playable SNES overworld**, much closer to **Super Mario World map logic** than a modern painted fantasy map.

This is **not** a brochure illustration and **not** a generic mobile game island.  
It should feel like a **16-bit inspired world map** with readable terrain, iconic landmarks, and a playful retro-game sense of place.

---

## Core Direction

### Target vibe
- Super Nintendo overworld energy
- Playable retro world map
- Bright, iconic, toy-like terrain
- Clean route readability
- Chunky cliff shelves and plateaus
- Simple, memorable landmark silhouettes
- Slightly top-down overworld composition

### Avoid
- Painterly fantasy island rendering
- Modern mobile-map pins as the primary visual language
- Realistic cliffs, foliage, or coastline treatment
- Too much environmental detail
- Soft concept-art shading
- Mismatched sprite/environment styles

---

## Main Corrections To Make

### 1. Add a small cottage at Home
Home should be represented by a **small iconic cottage/house**.

Requirements:
- Blue roof preferred, unless another color reads better in the palette
- Tiny overworld-style house
- Very simple silhouette
- Door and maybe one window
- Feels like a classic map landmark, not a realistic building
- Centered at the Home location

### 2. Push the whole map closer to SNES overworld logic
The map should look like:
- broad plateaus
- chunky cliff bands
- symbolic trees and bushes
- strong readable path routing
- landmark-based destinations

Not like:
- a painted island scene
- a modern fantasy mobile-map
- a cinematic environment illustration

### 3. Make the terrain simpler and more iconic
Terrain should use:
- flatter color blocks
- simple shadowing
- clear cliff shelf edges
- playful rounded hill masses
- deliberate destination plateaus

Reduce realism and decorative detail significantly.

### 4. Make landmarks do the identity work
Destinations should read primarily through landmark/environment design.

Use this direction:
- **Home** = small cottage
- **Projects** = workshop / lab / tower / build-space landmark
- **Experience** = themed attraction facade / gate / stage / portal
- **Awards & Creds** = trophy monument / castle / hall / pedestal
- **Contact** = small outpost / mailbox / dock / tiny hut

You can still use labels, but the landmark should be doing most of the communication.

### 5. Make the player character feel like it belongs
Right now the character can easily feel pasted into the world if the environment is too modern or painterly.

The map style should shift toward:
- retro sprite logic
- tile-map readability
- iconic rather than realistic terrain

The world should meet the character, not the other way around.

---

## Composition Rules

### Island structure
- One cohesive island map
- Clear coastline
- Broad central area for Home
- Stronger destination plateaus for major nodes
- Smaller side pockets for secondary nodes
- Paths that feel like an explorable overworld route

### Hierarchy
- **Home** is the central hub
- **Experience** and **Projects** should feel like major destinations
- **Contact** and **Awards & Creds** should feel like smaller side destinations

### Terrain hierarchy
- Fewer shapes
- Bigger, cleaner landform decisions
- Less noise
- No excessive perimeter contour rings
- No random equal-weight blobs

---

## Visual Style Notes

### Preferred look
- SNES-inspired overworld
- Bright flat colors
- Selective simple shading
- Chunky readable cliffs
- Clean path shapes
- Symbolic trees and bushes
- A little playful charm
- Feels like a map you can move around on

### Not preferred
- Overly detailed painterly foliage
- Realistic terrain rendering
- Generic fantasy map style
- Too much polish that loses the retro read
- UI pins dominating the landmarks

---

## UX / Readability Notes
- The route between nodes should stay very clear
- Each destination should be readable quickly
- The map should feel explorable in 2 seconds
- Prioritize **playable map readability** over decorative detail
- The final impression should be:  
  **"This feels like a retro game world map for TJ’s portfolio."**
